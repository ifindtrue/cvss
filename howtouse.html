zindex , opacity , cursor 같은 속성은 타입에 상관없이 모두적용되는 속성입니다.

타입 cvss의 객체는 block,path,circle,text로 나뉩니다.

각 타입별로 적용될 수 있는 속성값이 다를 수 있습니다.
현재 0.1버전에서는 path,circle,text의 개발이 충분히 되지않았습니다.

cvss의 모든속성의글자들은 대소문자를 구별하여야합니다.
cvss는 기본적으로 낙타표기법을 사용하고있습니다.

block :가장 기본적인 elementType

block 타입은 type 속성이 입력되지않았을 때 자동으로 적용되는 cvss는 가장 기본적인 속성입니다.

background :객체의 배경색 설정

var sample1=new Canvas("#sample1");
sample1.createElement("box",{type:"block",background:"orange"})

.createElement({element1:{...},element2:{...}});의 형식도 가능

x,y :좌표설정

var sample2=new Canvas("#sample2");
sample2.createElement("box",{type:"block",background:"orange",x:50,y:50});

x,y 속성을 통해 엘레먼트의 위치를 지정해줄수 있습니다. 또 block과 circle에 적용되는 origin 속성은 주로 퍼센테이지(일반정수도 가능)을 통해 x,y좌표에 객체의 어느부분이 위치할 것인지 지정해줄수 있습니다.

또한 x,y속성에 +=,-= 같은 증감연산자를 사용하여 기존의 값을 증감 할 수 있습니다.


x+10
x-10
var sample3=new Canvas("#sample3");
sample3.createElement("box",{type:"block",background:"orange"});

document.getElementById("sample3Btn").addEventListener("click",function(){
    sample3.element("box").cvss("x","+=10px");
});
document.getElementById("sample3Btn2").addEventListener("click",function(){
    sample3.element("box").cvss("x","-=10px");
});

참고로 .element(객체이름)를 통해 element를 지정하는 하는 것이 가능하고, 그 뒤에 바로 온 .cvss(속성이름,값);으로 지정한 element의 속성을 제어할 수 있습니다.
.cvss는 .cvss(속성이름,값); 형식뿐만아니라 .cvss({속석이름:값,속성이름:값...});와 같은 형식을 통해 일괄처리도 가능합니다.

width,height :가로길이,세로길이 설정

var sample4=new Canvas("#sample4");
sample4.createElement("box",{type:"block",background:"red",width:50,height:"100%"});

이 속성도 증감연산자(+=,-=)가 적용될 수 있습니다.
또 흥미로운것은 canvas요소의 가로,세로길이에 대해 퍼센이지가 적용될 수있다는 점입니다.
maxWidth,minWidth,maxHeight,minHeight 를 통하여 최대최소길이를 지정가능 합니다.

backgroundImage :이미지를 불러옵니다.

var sample5=new Canvas("#sample5");
sample5.createElement("object",{
    type:"box",width:480,height:60,onlyBaseElement:true,
    background:"#aaa",backgroundImage:"https://s-media-cache-ak0.pinimg.com/736x/8f/ca/9a/8fca9ac647f3f3f046b244c5aa22be07.jpg",border:"1px solid #ddd",
    rotate:-10,rotateOriginX:"50%",rotateOriginY:"50%"
});
sample5.element("object").hover(function(e,key){
    this.element(key).cvss("border","1px solid red");
    toggle*=-1;
},function(e,key){
    this.element(key).cvss("border","none");
    toggle*=-1;
});

for (i=1; i<=4; i++) {
    sample5.createElement("sprite"+i,{baseElement:"object",x:10,y:30+200*(i-1),backgroundImageY:-60*(i-1)});
    sample5.createElement("sprite"+i+"_moving",{baseElement:"object",x:10,y:100+200*(i-1),backgroundImageY:-60*(i-1)});
}
setInterval(function(){spriteMotion("sprite1_moving",{width:60,height:0});},300);
setInterval(function(){spriteMotion("sprite2_moving",{width:60,height:0});},300);
setInterval(function(){spriteMotion("sprite3_moving",{width:60,height:0});},300);
setInterval(function(){spriteMotion("sprite4_moving",{width:60,height:0});},300);

var toggle=-1;
function spriteMotion(key,sprite){
    var elementInfo=sample5.element(key).getInfo();
    if(elementInfo.backgroundImageX*toggle+sprite.width>=elementInfo.$backgroundImage.width){
        sample5.element(key).cvss('backgroundImageX',0);
        return 0;
    }
    sample5.element(key).cvss({backgroundImageX:"+="+toggle*sprite.width+"px"});
}

backgroundImage:"src repeatType x y" 의 형식을 취하고 있는 이 속성은, src의 값에 이미지의 링크를,
repeatType(no-repeat,repeat-x,repeat-y,repeat)은 블록내에서 이미지의 출력패턴의 결정을,
x,y는 박스내에서 출력된 이미지의 좌표를 결정합니다. (증감연산자 사용 가능)
덭붙혀 설명하자면 sec,repeatType,x와y의 값은 각각 backgroundImageSrc,backgroundImagePattern,backgroundImageX 와 backgroundImageY라는 속성으로 따로 관리가능합니다.
또 이미지가 로드되면 element속성에 $backgroundImage 라는 속성이 생기는데 이속성의 값을 통해 불러온 이미지의 실제 크기를 알 수 있습니다.

border :테두리 설정

var sample6=new Canvas("#sample6");
sample6.createElement("box",{type:"block",x:10,y:10,border:"5px solid orange"});

background:"px borderType borderColor" 형식으로 px,borderColor 은 각각 borderWidth(증감연산자 가능),borderColor라는 속성으로 따로 관리가 가능합니다.

rotate :회전

rotate:각도의 형식으로 rotateOriginX,rotateOriginY을 중심으로 지정한각도만큼 회전시킨다.
rotateOriginX,rotateOriginY는 퍼센테이지를 적용할수있는데 두속성다 50%의 값이라면 사각형의 중심을 기준으로 회전을한다.

path :폴리곤을 그릴 수 있는 타입

line :선긋기

var sample7=new Canvas("#sample7");
sample7.createElement("line",{type:"path",line:[[0,0],[30,20],[100,100]],background:"orange",borderColor:"red"});

line속성은 2차원 배열형식이고 [점][x,y] 이런식으로 구성되어 있다.
테두리 색과 굵기는 border 즉 borderWidth,borderColor 을 통해 지정가능하다.

circle :원과 타원을 그릴 수 있는 타입

r :원의 기본 요소

var sample8=new Canvas("#sample8");
sample8.createElement("circle1",{type:"circle",x:10,y:0,r:"10",background:"orange",origin:"0% -50%"});
sample8.createElement("circle2",{type:"circle",x:30,y:0,r:"10 30",background:"orange",origin:"0% -50%"});
sample8.createElement("circle3",{type:"circle",x:50,y:0,r:"10",r2:50,background:"orange",origin:"0% -50%"});

r은 r1,r2 로 나뉘는데 r1은 가로 반지름 r2는 세로반지름이다. 이 속성들을 통해 타원을 만들 수 있다.

text :글자를 적을 수 있는 타입

content :글자내용

var sample9=new Canvas("#sample9");
sample9.createElement("text",{type:"text",content:"텍스트",fontFamily:"굴림",fontSize:30,x:10,y:50});

이벤트

엘레먼트 이벤트속성에는 click,dblclick,dragstart,drag,drop,hover,mousein,mouseout,mousedown,mouseup 가있다.
사용방법은 canvas.element(객체이름).eventType(function(이벤트객체,이벤트엘레먼트이름){}/*hoverevent*/,function(){});
캔버스 자체의 이벤트타입에는 mousemove,mousedown,mouseup 정도가있는데
사용방법은 canvas.element(canvas).eventType=function(e){} 와 같다. 

var sample10=new Canvas("#sample10");
sample10.createElement("path",{type:"path",line:[[10,20],[300,30],[20,100],[50,30]],cursor:"pointer"});
sample10.element("path").hover(function(e,key){
    sample10.element(key).cvss({background:"red"});
},function(e,key){
    sample10.element(key).cvss({background:"none"});
});
